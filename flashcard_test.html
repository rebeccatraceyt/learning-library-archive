CSS Style Rules

CSS (Cascading Style Sheets) style rules are sets of instructions that define the visual presentation and layout of HTML elements on a web page.
They control properties such as colors, fonts, spacing, and positioning, enabling us to create consistent and visually appealing designs.
By applying these rules to HTML elements using selectors, we can apply specific styles to individual elements or groups, ensuring a separation between content and presentation for a better user experience.
These flashcards delve into the foundational aspects of the style rule, unveiling the importance of selectors, declarations, property/value pairs, and their impact.

* Note: Flashcards are optimised for larger screens due to their compact design.

INTERACTIVE

Selecting Elements in CSS

body {

/* Property: Value; */
color: red;

}
The selector (body) determines which elements receive the defined style.

Constructing Declarations

selector {

/* Style Declaration */
property1: value1;

}
Declarations, enclosed in curly brackets, follow selectors and hold property/value pairs.

Defining Properties and Values

selector {

/* Pairing */
property: value;

}
Declarations in CSS are composed of property/value pairs, defining precise stylistic attributes for elements.

Composing Property/Value Pairs

selector {

/* Property: Value; */
color: red;

}
Properties (e.g., color) refer to HTML element attributes affected by assigned values (e.g., red).
