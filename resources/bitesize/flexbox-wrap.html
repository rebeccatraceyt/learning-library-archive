Flexbox

Using flex-wrap

Wrapping is the movement of content to the next line (or row) when it runs out of room to fit on the screen horizontally. Wrapping is an essential part of any responsive design - without it, you’ll very quickly end up with squashed elements and a poor user experience. Wrapping is extremely important when it comes to designing for phone screens. Thankfully, the flexbox model can handle wrapping for us!

As with flex-direction, we have some options when it comes to flex wrapping. These are:

nowrap – The flex items will not wrap to the next row when the container shrinks in size (default option)
wrap – The flex items will wrap to the next row when they run out of space
wrap-reverse – The flex items will wrap to the next row in reverse order

Let's take a look at the flex-wrap property in action. We'll be working with a flex container which contains a total of twelve <div> elements, in order to give us enough content to see the wrapping in action. We'll give the flex-wrap property a value of wrap, which will cause the flex items to wrap to the next line when they run out of space.

.flex-wrap {
display: flex;
flex-wrap: wrap;
background-color: rgb(35, 187, 187);
}
