Flexbox

Creating a flexible container

So, you're sold on flexbox but not sure how to use it? No worries, let's dive in and take a look!

To start, let's assume we have this simple boilerplate HTML code:

<!DOCTYPE html>
<html>
<head>
<link href=”flex_basics.css” rel=”stylesheet”>
<title> Flexbox Basics </title>
</head>
<body>
<div class=”flex-container”>
<div>1</div>
<div>2</div>
<div>3</div>
</div>
</body>
</html>

To make an element a flex container, it's as simple as setting a flex display:

.flex-container {
display: flex;
background-color: rgb(35, 187, 187);
}

It should be noted that when we set an element to be a flex container, all of its child elements automatically become flex items. So, in this case, the three <div> elements (1, 2, 3) will become flex items that will automatically be resized and ordered by our flex container.
