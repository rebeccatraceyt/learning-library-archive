Flexbox

Using justify-content

Justifying content in a flex container allows you to align your flex items in a number of ways, similar to justifying text in a text document. There are a number of useful options here, but the most widely used are:

flex-start – The flex items are positioned at the start of the container (default option)
flex-end – The flex items are positioned at the end of the container
center – The flex items are centered in the container along the main axis
space-between – The flex items will have space between them, but there will be no space before the first item or after the last item
space-around – All flex items will have space both before and after them - results in first and last item being closer to the edges
space-evenly – All flex items will have an equal amount of space on both sides - first and last item will not be closer to the edges

As an example, let’s take a look at centering. We’ll be using the justify-content property here, and giving it a value of center:

.flex-wrap {
display: flex;
justify-content: center;
background-color: rgb(35, 187, 187);
}
