Python List Comprehension

Conditions in comprehensions

We’ve seen the power of expressions in list comprehensions, but we can also use conditionals. For example, if we want result to only contain elements which meet a certain criterion, we can specify this in our comprehension. Let’s see this in action!

Let’s have a look at the ‘names’ list from our first example. This time around, rather than just copying all the names to a new list, we only want to copy the ones that don’t start with the letter ‘T.’

Here’s an approach using a basic ‘for’ loop:

names = ['Barry, 'Anika', 'Zuri', 'Tomas']
new_names_list = []

for name in names:
if name[0] != 'T':
new_names_list.append(name)

print(new_names_list)

All we’re doing is running through the list of names, checking if the first letter is ‘T’ and, if it isn’t, adding it to the new names list. As expected, we get the same list of names, just without Tomas (sorry, Tomas).

['Barry, 'Anika', 'Zuri']

So, how would we approach this with a list comprehension? We know the basic structure from our previous ‘names’ example, but where would the conditional statement fit in? The list comprehension solution looks like this:

names = ['Barry, 'Anika', 'Zuri', 'Tomas']

new_names_list = [ name for name in names if name[0] != 'T']

print(new_names_list)

The conditional statement comes after the name of the list that will be iterated. This takes the form:

for member in iterable if member meets condition

We took a ‘for’ loop that was three lines in length and condensed it down into a single line of code. Pretty neat!

There is one important caveat to cover here. Our ‘if’ statement appeared at the end of the comprehension because we simply wanted to check if a value equalled something (a ‘T’ in this case). What would we do if we wanted to change a value in the list (rather than just omit it) based on whether a certain condition was met? The answer – we’d put the ‘if/else’ condition immediately after the expression instead:

numbers = [1, 22, 5, 65, 23, 42, 8]

new_nums_list = [ i if i > 10 else 0 for i in numbers ]

print(new_nums_list)

Here, we are looking to run through the list of numbers and, if the number is greater than 10, append it to the new nums list. However, if the number isn’t greater than 10, we want to change the number to 0 instead, then append the 0 to the list. Let’s see the output for a better idea of what that looks like:

[0, 22, 0, 65, 23, 42, 0]

As we can see, the 1, 5 and 8 from the original list have been changed to 0 in the new list because they were less than 10, as the ‘if’ statement specified.

To summarise: if you’re just checking values, the conditional check goes at the end of your list comprehension; if you are looking to change values based on the conditional, it goes after the expression.
