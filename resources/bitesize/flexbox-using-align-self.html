Using Align-Self

The align-self property does almost exactly the same thing that the align-items property does on flex containers. The difference is that align-items will affect all of the flex items within the flex container, whereas align-self can be used to set the alignment of specific flex items.

We won’t go in-depth with this property as we’ve covered most of the concepts on the align-items card, but we will briefly set the value on the flex item(s) we want to align:

As always, we’ll set this property on one of our elements directly in the HTML:

<div class=”flex-container-fixed-height”>
<div style="align-self: flex-end;">1</div>
<div>2</div>
<div style="align-self: center;">3</div>
<div>4</div>
</div>

The output to that will be an item centrally aligned on the cross axis:

2

3

4

This property has most use when working with either a fixed size container, or items that have differing heights.

For the purposes of this demo we have set the container min-height to 120px.

You can experiment with align-self in the the browser inspector tools, editing the values given to the style in the HTML!
