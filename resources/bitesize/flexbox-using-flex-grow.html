Flexbox

Using flex-grow

With the flex-grow property, we can set how much a flex item can grow relative to the other flex items around it. What does that mean exactly? Well, let’s jump straight into our walkthrough in order to visualise this.

First, we’ll set the flex-grow property on each element like so. Again, we’ll do this using inline CSS for clarity. The values we’ll be setting for this property are positive integers. The higher the number you set, the more space the element will have to grow.

<div class=”flex-container”>
<div style="flex-grow: 1;">1</div>
<div style="flex-grow: 1;">2</div>
<div style="flex-grow: 8;">3</div>
</div>

In the above code, the first two elements both have a flex-grow value of 1. This will give them equal space to grow. However, the third element has a value of 8, meaning it will have try to take up eight times the space that 1 or 2 take up. Let’s check the output of that code now:

2

3

The first two elements have an equal size, but the third has been given much more space to grow inside the container. Again, it will attempt to take up 8 times more space than item 1 or 2, but it may be constrained by the size of the flex container.

The number provided to the flex-grow property isn’t special or limited; it is simply the size relative to the other items. To highlight this we could set very large values such as this:

<div class=”flex-container”>
<div style="flex-grow: 1;">1</div>
<div style="flex-grow: 20;">2</div>
<div style="flex-grow: 90;">3</div>
</div>

Despite the size of these numbers, the output won’t be wildly large elements that take over the screen – they simply take the amount of container space relative to the number they have been given for their flex-grow property:

2

3

The container has remained the same size; it is simply the items within it that have divided up the amount of space they have to grow, depending on the numbers you set.
