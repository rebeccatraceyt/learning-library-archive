Env Vars Explained

GitPod environment variables
Gitpod allows you to set environment variables for your projects without having to use an env.py file. Try visiting: gitpod.io/variables Here, you can click on 'New Variable' to add an environment variable to your Gitpod environment.
Once 'New Variable' is clicked you can then add the relevant details in. In this example 'Name' is 'SECRET_KEY' and 'Value' is a random string.
You will also notice a third field when creating a variable: 'Scope'. This defines which repositories have access to that variable. The scope field takes the form of owner/repository, with 'owner' being the GitHub username and 'repository' being the name of the GitHub repo concerned.
