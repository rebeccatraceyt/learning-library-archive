Visual Hierarchy

Visual Hierarchy is a design principle in which the ordering of elements indicates their order of importance. It provides users with the right visual cues to lead them to a desired goal. A well-structured content hierarchy reinforces a natural reading pattern, creating a structured user experience, minimising uncertainty, and promoting interaction.

We can show importance by manipulating these text characteristics:
Size
Using larger element sizing for important content helps users to notice these elements more easily.
Color
Brighter colors typically catch the user’s eye much quicker than muted ones.
Contrast
Highly contrasting colors are much more eye-catching.
Alignment
Elements out of alignment can stand out over conventionally aligned ones.
Repetition
Using the same styling on multiple elements can signify that there is a relationship between their content.
Proximity
Grouping elements close together suggests that their content or purpose is related.
Whitespace
More space around an element can help draw the user’s eye towards it.
Texture and Style
Using a richer texture can help the element stand out over elements with flat texturing.
