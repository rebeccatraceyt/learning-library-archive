CSS Positioning

The CSS position property allows you to control the both the position of HTML content, and how that content affects the flow of surrounding content. Some values allow you to use the top, bottom, left, and right values to specify an offset from a default position.

There are five possible position values:

Static

This is the default value. The element is not positioned in any special way - it just follows the flow of the page, and is not affected by top, bottom, left, or right values.

Relative

The element is positioned where it would be with static positioning, but offset relative to itself by any values set for top, bottom, left or right. The space reserved for the element is the same as it would be with static positioning. Therefore, the offsets do not move other elements.

3

Absolute

The element is taken out of the flow of the document and there is no space reserved for it. The element is positioned relative to its nearest ancestor. If there is no ancestor, it uses the document body. The element scrolls with the ancestor element.

3

Fixed

The element is taken out of the flow of the document and there is no space reserved for it. The element is positioned relative to the viewport, and will stay in the same place even if the page is scrolled.
