Flexbox

Using align-items

Not to be confused with justify-content (which is for alignment on the main axis), align-items is used to align flex items along the cross axis. Combining both justify-content and align-items can lead to some very crisp layouts.

As always, we have a number of options to play with for this property. The main ones are:

stretch – The flex items will stretch in order to fill the available space on the cross axis (default option)
center – The flex items will be centered on the cross axis
flex-start – The items are positioned at the start of the cross axis
flex-end – The items are positioned at the end of the cross axis

Let's see this in action. We'll use the centre value for this example, and apply it to our align-items property:

.flex-wrap {
display: flex;
align-items: center;
background-color: rgb(35, 187, 187);
}

As we have not set a flex-direction, the default of row is being used. This means that our cross axis is the vertical axis. As we'd expect with this setup, our items have been centered vertically. While this might appear a bit garish, consider a situation where we have three side-by-side elements, all of which contain paragraphs of different lengths, yet all of which need to be the same height.

Trying to ensure that these elements are the same height with pure CSS would prove quite difficult, as we'd have to either set a max height and truncate the text, or dynamically determine the height of the largest and resize the others. However, with the stretch value applied to this property, we can achieve it in a single line of code. Not bad, eh?
