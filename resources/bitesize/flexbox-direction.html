Flexbox

Using flex-direction

By using the flex-direction property, we can set the main axis of our flex container. This will tell the container which direction we want the flex items to run in. We can choose from the following properties:

row – The flex items are displayed horizontally in a row (default option)
row-reverse – The same as row, but the items are in reverse order
column – The flex items are displayed vertically in a column
column-reverse – The same as column, but the items are in reverse order

As an example, let's take a look at the column layout by applying the flex-direction property to our CSS code and giving it the value of column:

.flex-container {
display: flex;
flex-direction: column;
background-color: rgb(35, 187, 187);
}
