Common Errors

IndexError - Index out of Range

Why am I getting this error when I index a list?

In Python, you'll get the error IndexError: list index out of range when you try to access the index of an item which doesn’t exist in a list.

For example, if there are three items in your list, and you try to retrieve item number 4, you will get this error, as there is no fourth item in your list to access.

In the example below that this error appears when trying to index the fourth item in a simple list:

dogs = ['terrier', 'labrador', 'retriever', 'sheep dog']
print(dogs[4])
user@ci-os /home/user (main) $ python index_error.py
Traceback (most recent call last):
File “index_error.py”, line 3, in <module>
print(dogs[4])
IndexError: list index out of range

Remember that list indexes start at 0. Therefore, the first item in the list has an index of 0. In that case, the list range runs from 0 to n - 1, where n is the number of items in the list. So, a list with four items is indexed from 0 to 3.

dogs = ['terrier', 'labrador', 'retriever', 'sheep dog']
print(dogs[0])
print(dogs[1])
print(dogs[2])
print(dogs[3])
user@ci-os /home/user (main) $ python index_error.py
terrier
labrador
retriever
sheep dog

The same concept applies to negative list indexing, but in reverse. Negative indexing ranges from -1 to -n, where n is again the number of items in the list.

In our example list, the negative index runs from -1 to -4, and starts from the last item in the list (i.e. the value of dogs[-1] is sheep dog).

Why am I getting this error when I use range with a for loop?

When you use the range function with a for loop, if you pass in just one parameter, the range will starts from 0 and increments up to one less than the number you have passed in. In the following example, the range of the list is 0 to 3. An error will be thrown if we try to loop through a range greater than this (0 to 4 in this example).

dogs = ['terrier', 'labrador', 'retriever', 'sheep dog']
for dog in range(5):
print(dogs[dog])

What if I don’t know how many items are in the list?

You can use the len() function to get the length any list. Just pass in the list as an argument, and the number of items in the list will be returned!

dogs = ['terrier', 'labrador', 'retriever', 'sheep dog']
for dog in range(len(dogs)):
print(dogs[dog])
user@ci-os /home/user (main) $ python index_error.py
terrier
labrador
retriever
sheep dog
