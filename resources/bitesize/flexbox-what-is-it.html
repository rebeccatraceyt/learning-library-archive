Flexbox

What is it?

Implementing an appealing and responsive layout in CSS historically required usage of one or more of the following techniques:

Block Display – for stacking elements on top of each other
Inline Block Display – for positioning elements side-by-side
Tables – simple multi-row & column layouts, at the cost of responsiveness
Positioning – absolute or relative positioning in relation to other elements
Floats – floats an element right or left within its parent container

None of these methods are optimal for designing the complex, responsive layouts we associate with modern websites.

Enter Flexbox

The need for a better solution led to the development of the CSS Flexible Box Layout, or Flexbox for short. Note that Flexbox is the name of a module, and not any one CSS property.

A simple flexbox layout requires a flex container. You can make any div a flex container by setting its display property to flex. Flex containers will dynamically modify both the size and ordering of their child elements to fill available space. We can control this behaviour by setting the following properties on the flex container:

flex-direction

Sets the direction that the child elements will be aligned in (horizontally or vertically). This is called the main axis. The axis perpendicular to this is called the cross axis.

Values: row, row-reverse, column, column-reverse

justify-content

Sets the alignment of child elements along the main axis. Generally used to define column spacing (dependent on main axis direction).

Values: flex-start, flex-end, center, space-between, space-around, space-evenly

align-items

Sets the alignment of child elements along the cross axis. Generally used to specify how rows should align (dependent on main axis direction).

Values: row, stretch, flex-start, flex-end, center

flex-wrap

If there are too many elements to fit on the screen, you can set this property to tell your flex container how to handle the overflows.

Values: nowrap, wrap, wrap-reverse

align-content

Specifies how to fill additional space on the cross axis if the child elements do not completely fill it.

Values: flex-start, flex-end, center, space-between, space-around, space-evenly, stretch
