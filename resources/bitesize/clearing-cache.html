Resolving Cache Issues

What is the browser cache?

Did you know that your web browser saves static assets from the sites you’ve visited into a temporary storage area called the cache? When you first visit a webpage, the site’s HTML, CSS, JS, and even some of the images will be saved into your browser’s cache.

Why? To speed up future loading - if you visit the same webpage multiple times, caching prevents the entire page having to be loaded each time, which saves on bandwidth for both you and the host of the website you’re visiting.

Remember: login cookies and localstorage data are not stored in the cache

Why you may need to clear the cache

The cache is a useful tool when combined with a completed, deployed website. However, it can cause trouble for a coder who is making frequent changes to their site’s static assets.

A common issue arises when a programmer updates their site’s CSS, but the changes do not display when they reload the webpage. This is because the browser has cached a previous version of the CSS, and has not loaded the new version.

If you have made updates to your codebase that are not appearing on your site, clearing the cache may resolve this.

How to empty cache and hard reload:

In this example, we will be using the recommended Browser, Google Chrome. For steps to follow on alternative browsers, you can use this helpful guide to Browser Cache.

1 Open Developer Tools by right-clicking on your mouse and selecting 'Inspect' or pressing the F12 button
2 Click and hold down the Reload button. From the drop-down select 'Empty Cache and Hard Reload'.

How to disable your cache:

Your browser’s Developer Tools will include an option to disable the cache. Note that this only applies when your Developer Tools are open. To do this:

1 Open your browser's Developer Tools
2 Navigate to the "Network" tab
3 Check the "Disable Cache" checkbox
4 Reload your page as normal (with Developer Tools still open).

Using incognito/private mode:

A quick way of bypassing your browser’s cache for testing purposes is to use an incognito (Chrome) or private (Firefox) window. These windows do not cache data, nor do they store cookies or browsing history. To open an incognito or private window:

Chrome:
PC: CTRL + SHIFT + N
MAC: CMD + SHIFT + N
Firefox:
PC: CTRL + SHIFT + P
MAC: CMD + SHIFT + P
