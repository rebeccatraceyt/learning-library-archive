Flexbox

Using Order

The ordering property allows us to position our items on screen in a different order to how they are written in the HTML code. In general, code in an HTML file is rendered from top-to-bottom, but with flexbox we can alter that as we please.

With our current HTML code, we would have the following output:

1

2

3

4

As expected, all the elements are in the order they are written in the code. Now, let’s apply some ordering to our flex items. For the purposes of this simple example, we will use inline CSS to set the ordering. This should ideally be done in the CSS file however, using classes to denote ordering:

<div class=”flex-container”>
<div style="order: 3;">1</div>
<div style="order: 2;">2</div>
<div style="order: 4;">3</div>
<div style="order: 1;">4</div>
</div>

According to the above ordering, we should see the element numbered 4 first (due to having the order of 1), then the element numbered 2, then 1, then 3. If we check our output:

1

2

3

4

We can see that is exactly what we ended up with! This property can be very useful when you need to reposition elements between larger and smaller screen sizes.
