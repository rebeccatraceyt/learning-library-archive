Flexbox

Using Flex-basis

Flex-basis sets the initial size of an item. An obvious question is “how does that differ from just setting a normal width on the item?” In short, unlike normal widths, the width set by flex-basis will still remain responsive across different screen sizes.

For example, if you set a fixed width of 200px on a normal element, it will remain that size, even if the screen becomes too small to display it without causing horizontal overflow. By using flex-basis, the element will remain 200px until the screen gets too small to contain it – then it will shrink down to fit the available space. Neat!

As always, we’ll set this property on one of our elements directly in the HTML:

<div class=”flex-container”>
<div>1</div>
<div>2</div>
<div style="flex-basis: 200px;">3</div>
<div>4</div>
</div>

Our output will then reflect this, with the third element being 200px wide:
