Sticky Headers and Footers

Sometimes you will want to keep either your header or footer visible at all times. Fortunately, this is a straightforward process which can be implemented using just a few lines of CSS!

For our header, we will create a new CSS rule which targets the HTML element with the id of header. We set the top property to 0 and the position property to sticky. This combination will cause the corresponding HTML element to stick to the top of the screen, irrespective of how far the user scrolls.

#header {
top: 0;
position: sticky;
}

Our footer rule will be created the same way, but we will change the top property to bottom, so it sticks to the bottom of the screen rather than the top.

#footer {
bottom: 0;
position: sticky;
}
