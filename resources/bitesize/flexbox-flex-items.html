Flexbox

What are flex items?

As mentioned previously, the elements within a flex container are called flex items. Elements within a flex container automatically become flex items - we don’t need to manually specify them. We have already covered some of the cool things a flex container can do by itself, so now let’s take a look at some of the neat stuff flex items can do.

As with flex containers, flex items have their own set of properties that can be used to create dynamic, responsive layouts. These are:

order – The order is which you want the flex items to appear on-screen. Default value is 0.
flex-grow – The same as row, but the items are in reverse order
flex-shrink – How much an item will shrink, relative to the other items around it. Default value is 1.
flex-basis – The initial size of an item before space is distributed. Default value is auto.
align-self – Overrides the default alignment of an item inside the container. Note: this overrides the align-items property set within the container itself if there is one

Alright, now we’ve covered what the options are, let’s dive into actually using them! We’ll use the following HTML code throughout our examples:

<!DOCTYPE html>
<html>
<head>
<link href=”flex_basics.css” rel=”stylesheet”>
<title> Flexbox Basics </title>
</head>
<body>
<div class=”flex-container”>
<div>1</div>
<div>2</div>
<div>3</div>
<div>4</div>
</div>
</body>
</html>
