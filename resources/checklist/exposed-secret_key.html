Checklist:

Exposed SECRET_KEY

If you’ve pushed a SECRET_KEY to Github, follow these steps:
import os
os.environ["SECRET_KEY"] = “your_super_secret_key_value_here”
import os
if os.path.exists("env.py"):
import env
SECRET_KEY = os.environ.get('SECRET_KEY')
Your SECRET_KEY is no longer exposed, and your new key is safe!
