Django 404 page creation

If a user tries to navigate to a page which doesn’t exist on your site, they will be shown a 404 page. By default, a plain 404 message is shown by Django when this happens. However, Django also does all the heavy lifting when it comes to custom error pages, so you can greatly improve your UX with a custom 404 page in a matter of minutes!
'DIRS': [os.path.join(BASE_DIR, 'templates')],
APPNAME/templates/404.html
{% extends "base.html" %}
{% block "content" %}
<h1>
This is my 404 page
</h1>
{% endblock "content" %}
Your customized 404 page will now load whenever you try to access a page that doesn’t exist!
