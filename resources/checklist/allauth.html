Installing and setting up AllAuth

Steps for installing and setting up the django-allauth package files:
pip3 install django-allauth==0.41.0
path('accounts/', include('allauth.urls')),
python3 manage.py makemigrations
python3 manage.py migrate
EMAIL_BACKEND = “django.core.mail.backends.console.EmailBackend"
To do that, run these few commands in the terminal:
Enter the python shell with python
Find the path to your allauth package with help(‘allauth’)
Copy that path into this command, to copy the files into your allauth folder:
cp -r [your-site-package-path]/allauth/templates/* ./templates/allauth/
Allauth is now set up and ready to use and customise in your Django project!
