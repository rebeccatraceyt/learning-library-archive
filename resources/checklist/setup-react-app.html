Setup a React Application

As React is an unopinionated library, there are a number of ways to structure a React application. In this checklist, we will cover the setup of a fully featured React App with industry-standard directory structure:
npx create-react-app . --template git+https://github.com/Code-Institute-Org/cra-template-moments.git --use-npm
npm start
npm install [package_name]@[version]
There is no set way to structure your application. You can include as many or as little of the adjacent directories based on your project's requirements, and the approach you feel best complements your development style.
Your new React app has successfully been created!
