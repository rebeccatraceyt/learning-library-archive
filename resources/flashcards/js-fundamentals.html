Functions and Scope

Vanilla functions

function greet(name) {
let message = `Hi, ${name}!`;
return message;
}

let result = greet(“Ada“);
console.log(result);
// Output: Hi, Ada!
The function is defined with the function keyword, followed by the function name (greet).
Inside the parentheses (), the parameter is specified (name).
Inside the curly braces {} the function body defines the code to be executed.
The return statement is the value the function will produce when called.
The function is called with a passing argument (Ada)
The result of the function call is stored in the result variable.

Arrow functions

const greet = (name) => {
let message = `Hi, ${name}!`;
return message;
}

let result = greet(“Ada“);
console.log(result);
// Output: Hi, Ada!
The const keyword defines a constant variable (greet) and assigns it an arrow function.
The function takes one parameter, inside the parentheses () (name).
Inside the curly braces {} the function body defines the code to be executed.
The return statement is the value the function will produce when called.
The function is called with a passing argument (Ada)
The result of the function call is stored in the result variable.

Single-Line Arrow functions

const addNum(a, b) => a + b;
let sum = addNum(6, 4);
console.log(sum);
// Output: 10
The const keyword defines a constant variable (addNum) and assigns it an arrow function.
The function takes two parameters, (a + b), inside the parentheses (), and returns their sum.
The function is called with two passing arguments (6, 4), then computes the sum of these values, storing in the sum variable.

Operators and Expressions

JavaScript Spread Operator

Without Spread Operator

let arr1 = [1, 2, 3];
let arr2 = arr1;
arr2.push(4);

console.log('array 1 = ', arr1);
// Output: array 1 = 1 2 3 4

console.log('array 2 = ', arr2);
// Output: array 2 = 1 2 3 4
In this example, arr1 and arr2 both refer to the same array
Any changes made to arr2 will also affect arr1, because they are essentially two references pointing to the same array.

JavaScript Spread Operator

With Spread Operator

let arr3 = [4, 5, 6];
let arr4 = [...arr3];
arr4.push(7);

console.log('array 3 = ', arr3);
// Output: array 3 = 4 5 6

console.log('array 4 = ', arr4);
// Output: array 4 = 4 5 6 7
In this example, arr3 and arr4 are two separate arrays.
The spread operator ([...arr3]) creates a new array (arr4), with the same elements as arr3, but they are stored in different memory locations.
This means that changes made to arr4 do not affect arr3. They are distinct arrays.
