What are CSS selectors?

/* Selector */

.title-text {
/* Property: Value; */

font-family: 'Montserrat';
font-weight: 400;
}
CSS Selectors are used to declare which HTML elements a particular style should be applied to
There are different types of selectors for targeting elements in different ways.
The types of selector we'll cover in this deck include Element Selector, Class Selector, ID selector, Pseudo-selector.

Element selectors

/* Selector */

p {
/* Property: Value; */

color: red;
font-weight: 400;
}
Element selecotrs target basic HTML elements by their tag type. For example, paragraphs and headings tags.
In the example the corresponding HTML you would be targeting would look like this:
<p>Hello I am Red!</p>
