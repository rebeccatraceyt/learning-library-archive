What are CSS selectors?
CSS Selectors are used to declare which HTML elements a particular style should be applied to

.title-text {
font-family: 'Montserrat', 'Sans-serif';
font-weight: 400;
}
Element selectors
Element selectors target basic HTML elements by their tag type. For example, paragraphs and headings.
Class selectors
Class selectors target multiple elements with a certain class. They are written using a full stop, then the class name.
ID Selectors
ID selectors target a single element with a certain id. They are written using a hash symbol, then the ID name.
Descendant selectors
Descendant selectors target elements in a hierarchical fashion. On the right, we target all list item elements which are descendants of unordered lists. Descendants of ordered lists are not affected.
Pseudo Selectors
Pseudo selectors target elements that are in a certain state of activity.
