CSS Selectors

CSS Selectors are the first part of a CSS rule, that determines which element from the HTML to apply a given style to.
Different selector types can be used, depending on whether you want to target mutiple elememts with the same style, a single element independently, or an element in a certain state of activity.

What are CSS selectors?

/* Selector */

.title-text {
/* Property: Value; */

font-family: 'Montserrat';
font-weight: 400;
}
CSS Selectors are used to declare which HTML elements a particular style should be applied to
There are different types of selectors for targeting elements in different ways.
The types of selector we'll cover in this deck include Element Selector, Class Selector, ID selector, Pseudo-selector.

Element selectors

/* Selector */

p {
/* Property: Value; */

color: red;
font-weight: 400;
}
Element selectors target basic HTML elements by their tag type. For example, paragraphs and headings tags.
In the example the corresponding HTML you would be targeting would look like this:
<p>Hello, I am Red!</p>
Now, any p element will be red:

Class selectors

/* Selector */

.my-class {
/* Property: Value; */

color: green;
}
Class selectors target multiple elements with a certain class.
They are written using a full stop, then the class name.
In the example the corresponding HTML you would be targeting would look like this:
<div class="my-class">
Hello, I am Green!
</div>
Now, any text within an element that has the class my-class will be green:

ID selectors

/* Selector */

#my-id {
/* Property: Value; */

color: blue;
}
ID selectors target a single element with a certain id.
In the example the corresponding HTML you would be targeting would look like this:
They are written using a hash symbol, then the ID name.
<div id="my-id">
Hello, I am Blue!
</div>
Now, any text within an element that has the id my-id will be blue:

Pseudo selectors

/* Selector */

i:hover {
/* Property: Value; */

color: green;
}
Pseudo selectors target elements that are in a certain state of activity.
They are written using the element of choice followed by a semi-colon, then the activity under which the style will apply.
In the example the corresponding HTML you would be targeting would look like this:
<i class="fab fa-github"></i>
Now, when the icon element is hovered over with the cursor, the icon colour will be green:

Descendant selectors

/* Selector */

ul li {
/* Property: Value; */

color: blue;
}
Descendant selectors target elements in a hierarchical fashion.
On the right, we target all list item elements which are descendants of unordered lists. Descendants of ordered lists are not affected.
In the example the corresponding HTML you would be targeting would look like this:
<ul>
<li>Hello, I am blue!</li> <li>Hello, I am blue too!</li>
</ul>
Now, any li child within the ul element will be blue:
