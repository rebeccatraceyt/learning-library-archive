CSS Style Rules

CSS (Cascading Style Sheets) style rules are sets of instructions that define the visual presentation and layout of HTML elements on a web page.
They control properties such as colors, fonts, spacing, and positioning, enabling us to create consistent and visually appealing designs.
By applying these rules to HTML elements using selectors, we can apply specific styles to individual elements or groups, ensuring a separation between content and presentation for a better user experience.

* Note: Flashcards are optimised for larger screens due to their compact design.

CSS Style Rules

/* Selector */
body {

/* Property: Value; */
color: red;

}
The selector (body) is used to determine which element(s) to apply the style to.
This is followed by one or more declarations, contained in curly brackets.
The declarations are composed of property/value pairs.
The property (color) refers to the property of the HTML element(s) that the value (red) will apply to.
Note the syntax of the declaration: a colon and space after the property with a semicolon after the value.
